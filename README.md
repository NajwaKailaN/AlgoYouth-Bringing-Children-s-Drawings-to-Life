# AlgoYouth: Bringing Childrens Drawings to Life
AlgoYouth is an interactive educational platform designed to make learning more engaging, creative, and meaningful for children.

Developed in collaboration with RAIA Kids Club, this project introduces children to the biodiversity of the Kerinci Rainforest through an immersive learning journey. Children explore native flora and fauna, create their own drawings, and watch their artwork come to life as animated rainforest characters.

By combining environmental education, storytelling, and technology, AlgoYouth transforms children from passive learners into active creators.
